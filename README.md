# Installation

```
mkdir ~/.my-zsh-scripts
cd ~/.my-zsh-scripts
git clone git@github.com:hubert-m/my-zsh-scripts.git .
```

Add to `~/.zshrc`

```
source ~/.my-zsh-scripts/.init-my-zsh-scripts
```
